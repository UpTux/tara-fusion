# TARA Fusion App Icon

This directory should contain the application icons:

## Required Icons

- **icon.icns** - macOS icon (512x512 or 1024x1024)
- **icon.ico** - Windows icon (256x256)  
- **icon.png** - Linux icon (512x512 or 1024x1024)

## Icon Guidelines

- Use high-resolution images (at least 512x512)
- Use transparent backgrounds
- Keep designs simple and recognizable at small sizes
- Follow platform-specific design guidelines:
  - macOS: Rounded corners, subtle shadows
  - Windows: Square with transparency
  - Linux: Various styles accepted

## Generating Icons

You can use tools like:
- `electron-icon-builder` npm package
- Online icon generators
- Image editing software (GIMP, Photoshop, etc.)

## Placeholder

Until proper icons are created, electron-builder will use default icons during development builds.
