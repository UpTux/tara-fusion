TARA Fusion
===========

Welcome to the TARA Fusion documentation!

Overview
--------

TARA Fusion is a comprehensive web-based and desktop application for conducting **Threat Analysis and Risk Assessment (TARA)** for security-critical systems. It combines traditional TARA methodologies with AI-powered threat intelligence, providing an integrated environment for security analysis, risk assessment, and documentation generation.

**Key Features:**

* **Visual Attack Tree Editor** - Create and visualize attack paths with an intuitive graphical interface
* **AI-Powered Threat Generation** - Leverage Google Gemini AI to discover potential threats
* **Comprehensive Risk Management** - Track threats, assets, damage scenarios, and mitigation strategies
* **Multi-User Collaboration** - Role-based access control for teams and organizations
* **Documentation Export** - Generate sphinx-needs compatible documentation
* **Cross-Platform** - Available as web application and desktop app (macOS, Windows, Linux)
* **Multi-Language Support** - Interface available in 6 languages
